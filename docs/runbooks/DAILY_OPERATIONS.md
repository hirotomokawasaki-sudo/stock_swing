# DAILY_OPERATIONS.md

## Pre-market checklist
- Confirm runtime mode
- Confirm API health
- Confirm broker connectivity
- Confirm latest macro calendar state
- Confirm source freshness
- Confirm no unresolved audit failures

## During market
- Monitor ingestion health
- Monitor decision generation count
- Monitor denied decisions for anomalies
- Monitor broker reconciliation

## Post-market
- Review decisions generated
- Review fills and reconciliation
- Archive raw snapshots as needed
- Produce daily summary report
