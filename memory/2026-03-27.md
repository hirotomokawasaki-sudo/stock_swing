## Key Design Patterns

### Safety Mechanisms
- **Raw Immutability**: Default fail on overwrite
- **Fail Closed**: Runtime mode, invalid stages → explicit errors
- **Paper-Safe Broker**: Live mode blocked at initialization
- **No Auto-Apply**: All parameter changes require operator approval
- **Kill Switch**: Emergency stop with fail-safe defaults
- **Audit Trail**: Comprehensive logging of all operations

### Key Learnings
- OpenClaw plugins must be in `openclaw/dist/extensions/` directory
- Global node_modules are NOT scanned automatically
- Symlink is required for externally installed plugins
- Python version (4.21.0) and npm version (1.15.0) are different - npm is the OpenClaw plugin

### Configuration Required After Detection
Once plugin is detected, set in `~/.openclaw/openclaw.json`:
